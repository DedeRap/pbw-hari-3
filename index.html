Manajemen Dataset

Aplikasi ini digunakan untuk mengelola dataset yang digunakan pada pemodelan perangkat lunak sehingga dapat ditelusur. Adapun kemampuan aplikasi ini sebagai berikut: 1. Membuat tabel dataset sesuai dengan definisi data yang telah dibuat di aplikasi Intelligence Creation 2. Mengisi/mengunggah dataset yang diperlukan dalam pemodelan perangkat lunak 3. Menampilkan statistic deskriptif dan kualitas data 4. Merekam perubahan dataset sebagai akibat hasil pemrosesan data 5. Merekam proyek mana saja yang menggunakan dataset ini
